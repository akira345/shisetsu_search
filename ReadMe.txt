勉強がてら作成した、施設検索システムです。

機能としては

PCでの利用：

GoogleMapを利用した施設の登録、コメント、登録済み施設への訪問履歴作成機能

携帯での利用：

GPS機能を利用した現在地の登録又は住所入力での登録。コメント、登録済み施設への訪問記録作成機能。

アバウトだけど周辺施設の検索機能とか。

施設検索システム設置方法
０：googlemapsAPIのキーを取得。GeoFormAPIのキーを取得
１：MySQLサーバ上でcreatetables.txtのSQLを実行し、テーブル作成
２：/lib/setting.phpを開く
３：DB接続文字列をセット
４：googlemapsAPI、GeoFormAPIのキーをセット。
５：GPS情報取得後の戻り先URLを設定します。（ktouroku.phpへの絶対パス）
６：文字コードに注意しながらサーバへFTP転送する。
７：MySQL上loginテーブルにユーザ名、PWを登録する。その際、PWはMD5でエンコードすること！！
　　（もしわからなければ、chk_md5.phpを利用するとよい）
以上。

動作環境
（あくまで開発者の動作環境です）
OS:CentOS5.2
PHP:5.2.6(一部の関数の関係で5.2以上でないと動かないとおもう）
MySQL:5.0.45
pear:MDB2 2.4.1
